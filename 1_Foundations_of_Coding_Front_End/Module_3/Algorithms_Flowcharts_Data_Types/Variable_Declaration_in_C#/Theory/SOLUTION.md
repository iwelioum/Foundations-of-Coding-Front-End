## Problem 1: Declaring and Initializing a Double Variable

### Solution

**Data Type:** Double is appropriate for storing decimal numbers.

**Variable Declaration:** We use the keyword `double` to declare the variable.

**Initialization:** Assign the value 49.99.

### Answer

```csharp
double productPrice = 49.99;
```

### Explanation of the Code

- `double` specifies that the data type is a double, which can store decimal values.
- `productPrice` is the name of the variable.
- `=` is the assignment operator.
- `49.99` is the decimal value assigned to `productPrice`.

## Problem 2: Declaring and Initializing a Boolean Variable

### Solution

**Data Type:** Boolean (`bool`) is appropriate for storing true/false values.

**Variable Declaration:** We use the keyword `bool` to declare the variable.

**Initialization:** Assign the value `true`.

### Answer

```csharp
bool isAdmin = true;
```

### Explanation of the Code

- `bool` specifies that the data type is a boolean, which can store true or false values.
- `isAdmin` is the name of the variable.
- `=` is the assignment operator.
- `true` is the boolean value assigned to `isAdmin`.
