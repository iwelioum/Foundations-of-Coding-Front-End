# Activity: Writing Pseudocode - Answer Key

## Problem 1: Calculate Total Marks

### Problem Statement

Write pseudocode to create a program that calculates the total marks for a student based on marks in three subjects: Math, Science, and English. The program should take input for the marks in each subject, calculate the total, and then print the total marks.

### Solution

#### Step 1: Define the Problem

Calculate the sum of marks from three subjects (Math, Science, English) and display the total.

#### Step 2: Identify Key Processes

- Get marks for Math
- Get marks for Science
- Get marks for English
- Add all three marks together
- Display the total

#### Step 3: Pseudocode
```
Start
Ask the user to enter the marks for Math
Ask the user to enter the marks for Science
Ask the user to enter the marks for English
Add the Math marks, Science marks, and English marks together
Display the total marks to the user
End
```

## Problem 2: Greet User by Name

### Problem Statement

Write pseudocode to create a program that takes a user's name as input and greets them with a message that says "Hello, [name]!".

### Solution

#### Step 1: Define the Problem

Take a user's name as input and display a personalized greeting message.

#### Step 2: Identify Key Processes

- Get the user's name
- Create a greeting message with the name
- Display the greeting

#### Step 3: Pseudocode
```
Start
Ask user for name
Set the greeting to "Hello, " plus name plus "!"
Display greeting to user
End
```
